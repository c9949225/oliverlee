Eclipse PDE Maven Plugin

This plugin is a Maven adapter of Eclipse PDE headless build automation. It generates and invokes PDE Ant build files to produce the final artifacts which can be installed/deployed to Maven local/remote repositories respectively.

Goals Overview

  • pde:pde: Build PDE artifact on the command line ( ie mvn pde:pde )
  • pde:attach: Build PDE artifact with Maven default build lifecycle, must be configured via a POM.
  • pde:ext: Build PDE artifact with custom build life cycle.
  • pde:test: Invoke test.xml Ant file if available at test phase
  • pde:clean: Clean the generated PDE files
