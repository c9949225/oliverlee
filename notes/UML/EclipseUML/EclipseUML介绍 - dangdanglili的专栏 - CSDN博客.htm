  EclipseUML介绍 收藏

　　Eclipse下的UML建模工具很多，但我一直没找到一款适合自己的UML建模工具，像Borland的together eclipse插件等都曾用过，但感觉不是很上手，今天在无意中找到EclipseUML，发现使用起来比较上手，现介绍介绍。

　　EclipseUML的下载地址为：http://www.eclipsedownload.com/，它分为Studio和Free版本，同时还分为j2ee和java版本，看大家的使用需要咯。Studio需要License，下载完后的EclipseUML只有一个jar文件：eclipseUML_E340_2008_studioEdition_JEE_3.4.1.v20081211.jar，那么如何安装呢？

　　打开Windows Command Line Console，输入:

java -jar eclipseUML_E340_2008_studioEdition_JEE_3.4.1.v20081211.jar

这样EclipseUML将会启动安装界面，然后再选择你的eclipse安装路径就可以啦，是不是很简单，呵呵。

　　安装完成后启动Eclipse，点击Help-->About Eclipse Platform查看是否安装成功，安装成功后会显示如下图所示的EclipseUML图标：

　　同时也会增加名为Modelinig的Perspective透视图，方便开发UML，下图为Modeling透视图的布局：
