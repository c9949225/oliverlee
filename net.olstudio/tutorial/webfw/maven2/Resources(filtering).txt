
简介:综合来说,两大功能

1-复制移动,将所有的resources目录下的文件移动到target/cass

2-字符串替换,根据Filtering

http://maven.apache.org/plugins/maven-resources-plugin/

    * Specifying a character encoding scheme
    * Specifying resource directories
    * Filtering
    * Including and excluding files and directories


1-src/main/resources/hello.txt:
	Hello ${name}
	
2-
<project>
  [...]
  <name>My Resources Plugin Practice Project</name>
  [...]
  <build>
    [...]
    <resources>
      [...]
      <resource>
        <directory>src/main/resources</directory>
      </resource>
      [...]
    </resources>
  [...]
  </build>
  [...]
</project>

3-mvn resources:resources

注:This will create a resource output in 
   target/class/hello.txt which contains exactly the same: Hello ${name}
   
4-
 <resource>
  	<directory>src/main/resources</directory>
  	-----------------------------------
  	<filtering>true</filtering>
  	-----------------------------------
  </resource>
  
5-mvn resources:resources 

	注:This will create a resource output in 
    target/class/hello.txt which contains:Hello My Resources Plugin Practice Project
    That's because the name variable was replaced by the value of the project's name (which was specified in the pom).
    
6-在命令行指定参数内容

mvn resources:resources -Dname="world"

	注:Hello world
	
7-设置自己的属性变量
	<project>
	  [...]
	  <properties>
	    <your.name>world</your.name>
	  </properties>
	  [...]
	</project>
	
8-把所有的自定义变量属性放在一个单独的文件

	<project>
	  [...]
	  <name>My Resources Plugin Practice Project</name>
	  [...]
	  <build>
	    [...]
	    <filters>
	      <filter>oliver.properties</filter>
	    </filters>
	    [...]
	  [...]
	  </build>
	  [...]
	</project>
