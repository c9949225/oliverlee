Ostriches!

Aren't these big birds cute? Ostriches can be up to 2.5 meters tall (8 ft) with a mass over 155 kilograms (320 lbs), plus they can run 65 km/h (40 mph). You might not want to mess with these babies!

Note: To view the tooltip effect in IE/Win browsers, you will need to download the csshover.htc file and save it in the same directory that this file is saved in.

The big image to the left is not a link, and yet Explorer for windows (IE5 and up) will still obey the :hover rules that make the popup possible, thanks to Peter Nederlof's wonderful Jscript. Woohoo!

Notice how when the pointer is moved off the image the popup vanishes, but when the pointer is moved off the image but remains on the popup, the popup is still there. This effect will make possible a simple CSS dropdown menu later on in the series.

Opera 7 Bug

This image was originally in a div that was floated to the left, but it so happens that if the element that serves as the popup "parent" is also floated, then Opera 7 will clip any part of the popup that lies across the float's margin (if any), and beyond that point it will not vanish once the popup parent is no longer hovered. Ick.

This bug can easily be prevented by floating an outer div container, and letting an inner non-floated div be the popup parent. Only when a single element is both floated and launches a popup, will this Opera 7 bug occur.

IE5.x/Win Bug

A problem crops up when viewing in IE5 or IE5.5 for Windows, with the result that the :hover rules are ignored. This only happens when the :hover rule selector does not contain the "type" of element being hovered. This means that if the popup parent is a div, the selector must read:

div.popup-parent:hover {styles}

and not just:

.popup-parent:hover {styles}

This also applies to the div.popup-parent:hover span {styles}. While this is a pain, it's not a fatal flaw when the bug is understood. We don't yet know if it is a basic bug in the older IE browsers, or if the Jscript just needs to be improved. If so, this problem may go away soon.
